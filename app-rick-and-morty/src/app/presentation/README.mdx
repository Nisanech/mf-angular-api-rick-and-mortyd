# Capa de Presentación

Implementa todo el software relacionado a los elementos visuales con los que el usuario final 
interactuará. Esto incluye los componentes de la interfaz de usuario, módulos, rutas, etc.

## Componentes

Elementos de la interfaz de usuario.

## Módulos

Agrupaciones de componentes y servicios para mantener la aplicación modular.

## Servicios de UI

Servicios específicos de la interfaz de usuario, como la gestión del estado de la UI o 
validaciones.
